# 🚀 Быстрый старт с OpenRouter (проще всего!)

OpenRouter - самый простой способ начать, не нужна кредитная карта!

## Шаг 1: Регистрация на OpenRouter (30 секунд)

1. Откройте https://openrouter.ai/
2. Нажмите "Sign up" (можно через Google)
3. Подтвердите email

**Готово!** Вы получите бесплатные кредиты для теста.

## Шаг 2: Получите API ключ (30 секунд)

1. Войдите на https://openrouter.ai/
2. Нажмите на аватар → "Keys"
3. Нажмите "Create Key"
4. Скопируйте ключ (начинается с `sk-or-v1-...`)

## Шаг 3: Настройте проект (1 минута)

```bash
# В папке spec-converter/backend:
cd backend

# Создайте config.py из шаблона
cp config_universal.py.example config.py

# Откройте config.py в любом текстовом редакторе
# Найдите раздел "ВАРИАНТ 2: OPENROUTER"
# Раскомментируйте (уберите #) эти строки:

API_PROVIDER = 'openrouter'
OPENROUTER_API_KEY = 'sk-or-v1-...'  # ← Вставьте ваш ключ
MODEL_NAME = 'anthropic/claude-sonnet-4'
```

## Шаг 4: Запустите (30 секунд)

### Windows:
Дважды кликните на `start.bat` в корне проекта

### Linux/Mac:
```bash
./start.sh
```

Вы увидите:
```
✅ Провайдер: openrouter
✅ Модель: anthropic/claude-sonnet-4
```

## Шаг 5: Используйте

1. Откройте в браузере `frontend/index.html`
2. Перетащите PDF файл
3. Нажмите "Конвертировать"
4. Готово!

---

## Почему OpenRouter?

✅ **Не нужна кредитная карта** - начните прямо сейчас  
✅ **Бесплатные кредиты** - протестируйте бесплатно  
✅ **Один ключ** - доступ ко всем моделям (Claude, GPT-4, Gemini)  
✅ **Дешевле** - часто ниже цены чем напрямую  
✅ **Простая регистрация** - через Google за 30 секунд  

---

## Стоимость

- **Бесплатные кредиты**: ~$1-2 при регистрации
- **Хватит на**: 50-100 страниц PDF
- **Далее**: ~2₽ за страницу
- **Пополнение**: от $5

## Альтернативные модели

В `config.py` можете выбрать:

```python
# Лучшее качество (дороже):
MODEL_NAME = 'anthropic/claude-opus-4'

# Баланс цена/качество (рекомендуется):
MODEL_NAME = 'anthropic/claude-sonnet-4'

# Альтернатива от OpenAI:
MODEL_NAME = 'openai/gpt-4-turbo'

# Самый дешевый (ниже точность):
MODEL_NAME = 'google/gemini-pro-1.5'
```

---

## Проблемы?

### "config.py не найден"
→ Убедитесь, что выполнили:
```bash
cp config_universal.py.example config.py
```

### "API key invalid"
→ Проверьте, что ключ начинается с `sk-or-v1-` и скопирован полностью

### "Rate limit exceeded"
→ Используйте бесплатные кредиты. Пополните баланс на openrouter.ai
