# 🔑 Выбор API провайдера

## Сравнение провайдеров

### 1. Anthropic API (оригинальный)

**Получить ключ**: https://console.anthropic.com/

**Плюсы:**
- ✅ Прямой доступ к Claude
- ✅ Официальная поддержка
- ✅ Максимальная стабильность

**Минусы:**
- ⚠️ Нужна кредитная карта
- ⚠️ Цена напрямую от Anthropic

**Цены:**
- Claude Sonnet 4: ~$3/1M вх., ~$15/1M вых.
- **~2₽ за страницу PDF**

---

### 2. OpenRouter (рекомендуется!) ⭐

**Получить ключ**: https://openrouter.ai/keys

**Плюсы:**
- ✅ Один ключ для всех моделей (Claude, GPT-4, Gemini)
- ✅ Не нужна кредитная карта для начала
- ✅ Часто дешевле чем напрямую
- ✅ Удобная панель управления
- ✅ Бесплатные кредиты для теста

**Минусы:**
- ⚠️ Иногда медленнее (зависит от нагрузки)
- ⚠️ Небольшая наценка на модели

**Цены через OpenRouter:**
- Claude Sonnet 4: ~$3/1M вх., ~$15/1M вых.
- GPT-4 Turbo: ~$10/1M вх., ~$30/1M вых.
- **~2-3₽ за страницу PDF**

**Доступные модели:**
```python
# В config.py можете выбрать:
MODEL_NAME = 'anthropic/claude-sonnet-4'      # Рекомендуется
MODEL_NAME = 'anthropic/claude-opus-4'        # Максимальная точность
MODEL_NAME = 'openai/gpt-4-turbo'             # Альтернатива
MODEL_NAME = 'google/gemini-pro-1.5'          # Дешевле
```

---

### 3. OpenAI API

**Получить ключ**: https://platform.openai.com/api-keys

**Плюсы:**
- ✅ GPT-4 Vision хорошо работает с таблицами
- ✅ Быстрый ответ
- ✅ Широкая поддержка

**Минусы:**
- ⚠️ Дороже чем Claude
- ⚠️ Чуть хуже с кириллицей
- ⚠️ Нужна кредитная карта

**Цены:**
- GPT-4 Turbo: ~$10/1M вх., ~$30/1M вых.
- **~3-5₽ за страницу PDF**

## Что выбрать?

### Для начала → OpenRouter ⭐
**Почему:**
- Не нужна кредитная карта
- Бесплатные кредиты для теста
- Можете попробовать разные модели
- Самый простой старт

### Для постоянного использования → Anthropic API
**Почему:**
- Прямой доступ = надежнее
- Предсказуемые цены
- Максимальная стабильность

### Для экспериментов → OpenRouter
**Почему:**
- Доступ ко всем моделям
- Легко сравнить результаты
- Гибкость выбора

## Как настроить

### Шаг 1: Выберите провайдера

Откройте `backend/config_universal.py.example`

### Шаг 2: Раскомментируйте нужный блок

#### Для Anthropic:
```python
API_PROVIDER = 'anthropic'
ANTHROPIC_API_KEY = 'sk-ant-api03-...'
MODEL_NAME = 'claude-sonnet-4-20250514'
```

#### Для OpenRouter (рекомендуется):
```python
API_PROVIDER = 'openrouter'
OPENROUTER_API_KEY = 'sk-or-v1-...'
MODEL_NAME = 'anthropic/claude-sonnet-4'
```

#### Для OpenAI:
```python
API_PROVIDER = 'openai'
OPENAI_API_KEY = 'sk-...'
MODEL_NAME = 'gpt-4-turbo'
```

### Шаг 3: Сохраните как config.py

```bash
cp config_universal.py.example config.py
nano config.py  # вставьте ваш ключ
```

---

## Регистрация на OpenRouter (самый простой вариант)

1. Откройте https://openrouter.ai/
2. Нажмите "Sign up" (можно через Google)
3. Перейдите в "Keys" → "Create Key"
4. Скопируйте ключ (начинается с `sk-or-v1-...`)
5. Вставьте в `config.py`:
   ```python
   API_PROVIDER = 'openrouter'
   OPENROUTER_API_KEY = 'sk-or-v1-ваш-ключ'
   MODEL_NAME = 'anthropic/claude-sonnet-4'
   ```

**Готово!** Можете начинать без кредитной карты.

## Проверка работы

После настройки проверьте:

```bash
cd backend
python app.py
```

Вы должны увидеть:
```
✅ Провайдер: openrouter
✅ Модель: anthropic/claude-sonnet-4
```

Если видите:
```

```

Значит config.py не найден или содержит ошибки.

## FAQ

**Q: Можно ли переключаться между провайдерами?**  
A: Да! Просто измените `API_PROVIDER` в config.py и перезапустите сервер.

**Q: Какой провайдер дешевле?**  
A: OpenRouter через Gemini (~1₽/стр), но точность ниже. Для баланса - Claude Sonnet 4.

**Q: Работает ли без API ключа?**  
A: Нет, для распознавания обязательно нужен AI API.

**Q: Можно ли использовать бесплатно?**  
A: OpenRouter дает бесплатные кредиты для теста (~$1-2), этого хватит на 50-100 страниц.
