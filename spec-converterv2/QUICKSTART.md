# 🚀 Быстрый старт за 3 минуты

## Шаг 1: Получите API ключ (1 минута)

1. Откройте https://console.anthropic.com/
2. Зарегистрируйтесь или войдите
3. Перейдите в "API Keys"
4. Создайте новый ключ
5. Скопируйте его (начинается с `sk-ant-...`)

## Шаг 2: Настройте проект (1 минута)

```bash
# В директории spec-converter выполните:

# Создайте config.py из шаблона
cd backend
cp config.py.example config.py

# Откройте config.py в текстовом редакторе
# Вставьте ваш API ключ:
ANTHROPIC_API_KEY = 'sk-ant-ваш-ключ'
```

## Шаг 3: Запустите (1 минута)

### Windows:
Просто дважды кликните на файл `start.bat`

### Linux/Mac:
```bash
./start.sh
```

### Или вручную:
```bash
cd backend
pip install -r requirements.txt
python app.py
```

## Шаг 4: Используйте

1. Откройте в браузере файл `frontend/index.html`
2. Перетащите PDF в окно
3. Нажмите "Конвертировать"
4. Готово! Файл скачается автоматически
