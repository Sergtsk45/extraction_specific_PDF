# 🔧 ИСПРАВЛЕНИЕ: Проблемы с конвертацией

## ❌ Что было неправильно

### Проблема 1: Нет заголовка в таблице
**Было:**
```
Строка 1: ['ВТ-3', 'Хозяйственно-питьевой водопровод...', ...]
Строка 2: [данные]
```

**Должно быть:**
```
Строка 1: ['Позиция', 'Наименование и техническая характеристика', ...]  ← ЗАГОЛОВОК
Строка 2: ['В1-3', 'Хозяйственно-питьевой водопровод...', ...]
```

### Проблема 2: Все данные в одном листе
**Было:**
- 1 лист "Спецификация" со всеми страницами

**Должно быть:**
- Лист "В1-3 Офисы стр.1"
- Лист "В1-3 Офисы стр.2"  
- Лист "Т3-Т4 Офисы"
- Лист "К1 Офисы стр.4"
- Лист "К1 Офисы стр.5"

### Проблема 3: Ошибки распознавания
**Было:**
- "ВТ-3" вместо "В1-3"
- "ЧВТ" вместо "УВ1"
- "ЧВТ В2" вместо "УВ1 В2"

**Причина:** AI путал кириллические буквы:
- В → ВТ
- У → Ч  
- 1 → Т

## ✅ Что исправлено

### 1. Улучшен промпт для AI

Добавлены критически важные инструкции:

```python
КРИТИЧЕСКИ ВАЖНО:
1. Первая строка ОБЯЗАТЕЛЬНО должна быть ЗАГОЛОВКОМ
2. Если видишь заголовок таблицы - извлеки его как первую строку
3. НЕ путай буквы: "В" это В, не "ВТ"; "У" это У, не "Ч"
4. Пустая позиция = "" (пустая строка)
5. Многострочный текст = объединить через пробел
```

### 2. Изменён формат ответа AI

**Было:**
```json
{
  "rows": [
    [данные без заголовка]
  ]
}
```

**Стало:**
```json
{
  "sheet_name": "В1-3 Офисы",
  "rows": [
    ["Позиция", "Наименование...", ...],  ← обязательный заголовок
    ["В1-3", "Водопровод...", ...],
    ...
  ]
}
```

### 3. Переписана функция создания Excel

**Было:**
```python
def create_excel_from_data(all_rows, output_path):
    # Свалить всё в один лист
    builder.create_sheet("Спецификация", all_rows)
```

**Стало:**
```python
def create_excel_from_pages(pages_data, output_path):
    # Группировать по секциям
    # Объединять страницы одной секции
    # Создавать отдельные листы
    sections = {}
    for page in pages_data:
        sheet_name = page['sheet_name']
        # Группируем "В1-3 стр.1" и "В1-3 стр.2" → "В1-3"
        base_name = sheet_name.split(' стр')[0]
        if base_name in sections:
            sections[base_name].extend(rows[1:])  # без дублирования заголовка
        else:
            sections[base_name] = rows
```
